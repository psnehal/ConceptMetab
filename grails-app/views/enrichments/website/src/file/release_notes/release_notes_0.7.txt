<p>
    This version expands the visual style options even more by letting you set images to nodes and
    draw edges with dotted or dashed lines. From now on you can also use just plain JavaScript objects to define a network,
    and completely avoid the XML formats, if you want. Exporting the network as SVG is now possible, too.
</p>

        <li><span class="bug_id">2397</span>: Dashed edge lines.</li>
        <li><span class="bug_id">2439</span>: Export the network as SVG.</li>
    </ul>
    <ul class="bug_fixes">
